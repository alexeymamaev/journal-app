# Иконки

Пока в этой папке лежит только README. Для PWA и «на главный экран» iOS нужны:

- `icon-180.png` — apple-touch-icon
- `icon-192.png` — manifest (фоновый)
- `icon-512.png` — manifest (splash)

Можно сгенерировать из любой квадратной картинки — поставлю как дойдут руки. Без них установка PWA работает, иконка будет дефолтная.
